# Testing Plan for Driver Database Project

## Goal
Test the complete integration of the driver database system.

## Features to Test

### 1. File Loading
- Load at least 100 driver records
- Verify correct loading

### 2. Insert Driver
- Insert at beginning, middle, end
- Verify correctness

### 3. Search Driver
- Search existing and non-existing drivers

### 4. Recent Licenses
- Input N
- Verify correct results

### 5. Oldest Licenses
- Input N
- Verify correct results

### 6. Move to Inactive
- Remove from active
- Add to inactive

### 7. Menu Interface
- All options work
- Invalid input handled

## Compilation Test
- Run make
- Ensure no errors
